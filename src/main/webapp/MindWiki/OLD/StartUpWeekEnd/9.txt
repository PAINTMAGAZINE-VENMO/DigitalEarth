NooPolis wird ein SocialNetwork, das eine virtuelle Ökonomie entstehen lässt. 

Die drei wichtigsten Werte von NooPolis sind 1. Transparenz, 2. Vertrauen und 3. Selbstorganisation. 

1. Transparenz entsteht durch die Verwendung von OpenSource-Software. Der Quelltext kann auf jeder Seite eingesehen werden. Auch die Administration der Server ist transparent. 

2. Vertrauen entsteht durch ein WebOfTrust. Die Identität aller User wird durch ein Netz von gegenseitigen Vertrauensbezeugungen verifiziert und gesichert. Auf diese Weise entsteht eine vertrauenswürdige Community.

3. Selbstorganisation entsteht durch BuergerBeteiligung. Verfassung. Haushalt.
