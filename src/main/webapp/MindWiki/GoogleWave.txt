!How to make a public wave:
# Go to CONTACTS (on lower left side of screen)
# "Add a new contact": public@a.gwave.com 
# __important__: ignore the message "User does not have a Google Wave account" - just add it anyhow (by pressing ENTER).
# "Add user or group to wave": Add the group "public"
# Now you can add this user to any GoogleWave you are in to make it a public one
